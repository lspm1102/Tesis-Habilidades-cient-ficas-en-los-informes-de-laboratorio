# Tesis: Habilidades científicas en los informes de laboratorio

Este repositorio reúne materiales asociados al análisis automatizado de habilidades científicas en informes de laboratorio, desarrollado en el marco de una tesis de maestría.

En particular, se incluyen las métricas obtenidas para distintos clasificadores y los *prompts* utilizados para la clasificación con LLM.

## Contenido del repositorio

El repositorio contiene:

- métricas de desempeño utilizadas para comparar los modelos;
- *prompts* empleados para la clasificación automática de enunciados;
- materiales complementarios vinculados al análisis de habilidades científicas en informes de laboratorio.

## Contexto del trabajo

El trabajo se centra en el análisis de informes elaborados por estudiantes de un curso introductorio de Física Experimental. El objetivo general es identificar y caracterizar habilidades científicas evidenciadas en las producciones escritas del estudiantado mediante técnicas de Inteligencia Artificial, en particular de Procesamiento del Lenguaje Natural.

Las clases consideradas para la clasificación de enunciados se vinculan con habilidades científicas asociadas al trabajo experimental, el análisis de datos, el análisis de gráficos, la evaluación de la incertidumbre y la evaluación del diseño o procedimiento experimental.

## Métricas

Las métricas incluidas permiten comparar el desempeño de diferentes enfoques de clasificación, entre ellos modelos clásicos de aprendizaje automático y modelos de lenguaje. 

Estas métricas fueron utilizadas para analizar el rendimiento de los clasificadores en cada una de las clases definidas y para seleccionar estrategias de clasificación aplicables al corpus completo de informes.

## Prompts

Se incluyen los *prompts* utilizados para orientar la clasificación automática de enunciados mediante modelos de lenguaje. Estos fueron diseñados a partir de la guía de anotación elaborada para el trabajo.
